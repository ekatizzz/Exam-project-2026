# Green Loop — AI-Assisted Eco Residential Complex
### Green Loop — концептуальный проект многофункционального жилого комплекса с вертикальным озеленением, разработанный с применением методов генеративного искусственного интеллекта и алгоритмов пространственной оптимизации.
### Проект демонстрирует применение ИИ на всех этапах архитектурного проектирования: от генерации концепции до анализа планировок, оптимизации потоков и детализации внутренней среды.
### Project Overview
### Тип проекта: Многофункциональный жилой комплекс (Mixed-use Eco Residential Complex)
### Площадь участка: 5 000 м²
### Общая площадь: ~21 000 м²
### Этажность: 7 этажей + подземный уровень
### Жилой фонд: 48 квартир
### Энергоэффективность: Класс A+
### Срок проектирования: 3 недели (вместо традиционных 14–18 недель)
